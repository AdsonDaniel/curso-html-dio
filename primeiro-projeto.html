Introdução

HTML é uma sigla em inglês que significa HyperText Markup Language, na tradução para o português significa LINGUAGEM DE MARCAÇÃO DE HIPERTEXTO.

É utilizada para criação de páginas web, sendo uns dos principais “mecanismos” para criação de um conteúdo para internet, onde é regida por TAGS. As tags são utilizados para informar ao navegador que tipo de estrutura está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

Qualquer conteúdo inserido, tem que ficar dentro das tags que são formadas pelos sinais ( < >), que significa que a tag está começando e ( < />), que significa que a tag finalizou, o conteúdo desejado fica dentro delas, como no ex: ( <>) adicionar conteúdo ( < />). E dentro dos sinais ( <> ), que agora chamamos de tag de abertura e tag de fechamento, é colocado o "nome" da tag, tanto na de abertura como na de fechamento.

Os “nomes” das tags que determinam como elas iram agir, cada tag tem uma função diferente, e o nome da tag pode ser dado por uma letras, siglas ou nomes. Por exemplo, para escrever um parágrafo é utilizado uma tag com a letra “ p” dentro da tag. Ex: (<"p">) conteúdo do parágrafo (< p/> ).

Principais tags

Como existe centenas de tags diferentes, irei comentar somente as três principai que formam a estrutura base, sao elas:

  1. Tag html - é o elemento raiz que define todo o documento HTML.
  2. Tag head - contém metadados sobre o título e a codificação de caracteres da página.
  3. Tag body - inclui todo o conteúdo que aparece na página.
    OBS: Lembamos que os nomes devem ficar dentro da tag <>

Atributos (voltar)

E algumas tags vem acompanhadas de atributos, que as vezes são obrigatórios ou opcionias. São palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. E esses atributos podem recebr valores, modificando ainda mais o comportamento daquela tag.
